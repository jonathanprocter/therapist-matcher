# Therapist Matcher

An interactive self-assessment that helps you discover which type of therapy and therapist might be the best fit for you. The entire application runs client side in the browser.

## Features

- 18 question survey with a progress indicator
- Calculates scores for several therapeutic approaches
- Generates a personalized narrative and list of therapist qualities
- Works completely offline – just open the HTML file

## Running Locally

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.

That's it! There is no server component or build step required.

## Deployment

Because everything is contained in `index.html`, the project can be deployed to any static web hosting service (e.g. GitHub Pages, Netlify, or a simple web server). For GitHub Pages:

1. Push this repository to GitHub.
2. Enable Pages in the repository settings and choose the main branch.
3. Your site will be available at the URL provided by GitHub.
